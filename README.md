# plat2cad.com

Landing page for [PLAT2CAD](https://plat2cad.com) — CAD drafting support for
land-survey, right-of-way, and title teams: recorded plats and deeds redrawn
into layered DWG, with closure reports and COGO point files.

Static site, no build step; `index.html` is the whole thing. Deployed on
GitHub Pages at the apex domain (see `CNAME`).

Questions or a plat to send: **plats@plat2cad.com** · first sheet free.
